## 📁 Projects

### Know The World

Is a web application designed to help users explore countries around the globe, access detailed country information and plan their travel. It includes user authentication features to provide a personalized experience.

Framework used is React for the frontend. I was the group leader of this Moringa Phase 2 final project and we used the Agile methodology. My roles were creating the:

- Homepage
- Individual continent country page
- Routes

🔗 [GitHub Repo](https://github.com/Moringa-SDF-PT10/Group-2-Know-The-World-Project)

🚀 [Live Demo](https://group-2-know-the-world.netlify.app/)

### EcoVibe Kenya

Ecovibe Kenya provides consultancy services and keeps up to date with evolving ESG landscape. The web application is the portal that connects the clients to the company. 

The frameworks used are React for frontend and Flask for backend. It was a group project and we used agile methodology. My roles were creating the:

- ERD diagram
- DB models for the backend
- About us page
- Navigation bar
- Style the single blog page
- Service page (both frontend and backend) for both the client and admin side

Project is under NDA hence can't share the Github repo

🚀 [Live Demo](https://www.ecovibe.co.ke/)
